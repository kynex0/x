X Takipçi Botu API Dokümantasyonu

Bu API, X Takipçi Botunu programatik olarak kontrol etmenizi sağlar. Aşağıda mevcut tüm API endpoint'lerini ve kullanımlarını bulabilirsiniz.

Durum Bilgisi

GET /api/status
Kimlik Doğrulama Gerektirmez

Botun mevcut durumunu ve yapılandırma bilgilerini döndürür.

Örnek Yanıt:
{
  "followedCount": 42,
  "config": {
    "username": "your_username",
    "maxFollowsPerDay": 50,
    "followDelayMin": 15,
    "followDelayMax": 30,
    "useProxy": false,
    "headless": false
  }
}

Takip Edilen Kullanıcılar

GET /api/followed-users
Kimlik Doğrulama Gerektirmez

Daha önce takip edilmiş kullanıcıların listesini döndürür.

Örnek Yanıt:
[
  "user1",
  "user2",
  "user3"
]

Yapılandırma Güncelleme

POST /api/config
Kimlik Doğrulama Gerektirmez

Bot yapılandırmasını günceller.

Parametre Listesi:
  • username: X kullanıcı adı
  • password: X şifresi
  • maxFollowsPerDay: Günlük maksimum takip sayısı
  • followDelayMin: Takipler arası minimum gecikme (saniye)
  • followDelayMax: Takipler arası maksimum gecikme (saniye)
  • headless: Tarayıcının arka planda çalışması için boolean değer
  • useProxy: Proxy kullanımını etkinleştirmek için boolean değer
  • proxyHost: Proxy sunucu adresi
  • proxyPort: Proxy port numarası
  • proxyUsername: Proxy kullanıcı adı (opsiyonel)
  • proxyPassword: Proxy şifresi (opsiyonel)
Örnek İstek:
POST /api/config
Content-Type: application/json

{
  "username": "your_username",
  "password": "your_password",
  "maxFollowsPerDay": 50,
  "followDelayMin": 15,
  "followDelayMax": 30,
  "headless": false,
  "useProxy": true,
  "proxyHost": "proxy.example.com",
  "proxyPort": "8080",
  "proxyUsername": "proxy_user",
  "proxyPassword": "proxy_pass"
}
Örnek Yanıt:
{
  "success": true,
  "message": "Yapılandırma güncellendi"
}

Bot Başlatma

POST /api/start
Kimlik Doğrulama Gerektirmez

Botu belirtilen hedef hesaplar için başlatır.

Parametre Listesi:
  • targetAccounts: Takipçileri takip edilecek X hesaplarının dizi listesi
Örnek İstek:
POST /api/start
Content-Type: application/json

{
  "targetAccounts": ["elonmusk", "jackdorsey"]
}
Örnek Yanıt:
{
  "success": true,
  "message": "Bot başlatıldı",
  "targetAccounts": ["elonmusk", "jackdorsey"]
}
